Simple Object Orientation 3

Modified the app from Simple Object Orientation 2 to use dependency injection to add flexibility.

GenericPet, Cat and Dog inherit from Pet abstract base class.

The makeNoise function is now injected and allows the noise to be specified at run time.

From a user point of view the app looks the same as yesterday, but it is different under the hood.

  • Enter a name and if you are creating a generic pet enter a species.
  • Then create the pet - either generic, dog or cat
  • Select the pet you want to interact with using the radio buttons
  • Generic pets use the makeNoise method from the Pet class
  • Generic Pets, Cats and Dogs are created with a makeNoise function which is injected at run time
  • (Only) Cats have a purr method
  • (Only) Dogs have a fetch method
