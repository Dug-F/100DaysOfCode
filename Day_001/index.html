Simple Object Orientation 1

Create simple object with property, set and retrieve value, manipulate with method

Use 'create object' button to create the object using the constructor with the initial value. This does not update the current pet type on the screen

Then use the 'Get current pet type' button to show the current contents of the object type property using the getter method.

Use the 'Set new pet type' button to update the object type property with a new value using the setter method. This does not update the current pet type on the screen.

Use the 'Get current pet type' button again to display the updated object type using the getter.

Finally, use the 'What kind?' button to use the object 'kind' method to return a modified version of the type.
